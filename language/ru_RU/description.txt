AStat - это плагин, который генерирует статистические отчеты гораздо более детально, чем статистика включенная в Piwigo по умолчанию.
Расширенные возможности включают "просмотренные страницы", "просмотренные изображения", "число IP адресов"; можно задавать период времени для анализа, можно выбирать определенные IP адреса, конкретные категории или конкретные изображения.


Плагин для работы требует предварительной установки плагина 'GrumPluginClasses 3.5.2'.
